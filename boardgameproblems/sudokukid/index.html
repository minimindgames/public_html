Sudoku 4x4 problem

The hardest Sudoku 4x4 has only four clues.

The goal of Sudoku 4x4 problem is to solve the hardest possible Sudoku 4x4 puzzle.

How to play

Solve Sudoku to raise a level and make a mistake to drop a level - then keep playing to raise again..

On each level, the difficulty increases because initial clue count decreases by one.

Sudoku 4x4 rules

Sudoku 4x4 is played on a grid of 4x4 cells, which has four columns, four rows and four boxes.

Initially, Sudoku is filled with some clues and you need to fill empty cells with correct symbols.

Sudoku is solved when each column, each row and each box contain exactly once each of the given four symbols.
