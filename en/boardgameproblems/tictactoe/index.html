Tic Tac Toe problem

Tic Tac Toe has six strategy levels.

The goal of Tic Tac Toe problem is to beat the hardest Tic Tac Toe strategy.

How to play

Win a game to raise a level and lose a game to drop a level - then keep playing to raise again.

Each game level corresponds to computer's strategy, that is, how many moves ahead computer plans. In order to win you obviously need to plan one move further.

At the sixth level, computer's strategy is perfect so the game is always a draw if you play without mistakes too.

Tic Tac Toe rules

Tic Tac Toe is a turn-based board game for two players, X and O. The game board is a 3x3 grid of squares, where the players alternatively mark one square. Winner is the first player to get a horizontal, vertical or diagonal row of three squares.

Misere Tic Tac Toe

Lose Tic Tac Toe to win Misere.

Misere is anti/reverse Tic Tac Toe, where you win if you can enforce your opponent to make three in a row.

How to play

Win a game to raise a level and lose a game to drop a level - then keep playing to raise again.

Each game level corresponds to computer's strategy, that is, how many moves ahead computer plans. In order to win you obviously need to plan one move further.

Misere Tic Tac Toe rules

Misere Tic Tac Toe is a turn-based board game for two players, X and O. The game board is a 3x3 grid of squares, where the players alternatively mark one square. Winner is the player who enforces opponent to get a horizontal, vertical or diagonal row of three squares.
