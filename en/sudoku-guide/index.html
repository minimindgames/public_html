Sudoku guide: Sudoku puzzle and how to solve Sudoku

Sudoku has ONE CORRECT SOLUTION which can be reached by logic without any guessing.

Sudoku puzzle

Sudoku puzzle is a grid of 9x9 cells with nine columns, nine rows and nine boxes, as illustrated in the picture.

Sudoku puzzle is solved when each column, each row and each box contain exactly once numbers between 1 and 9.

Sudoku is a charmingly simple puzzle :)

How to solve Sudoku

Initially, a Sudoku puzzle is prefilled with about twenty clue numbers, while all the other cells are empty.

To solve Sudoku you need to fill the empty cells with numbers 1 to 9, which is simpler said than done, as each cell has only one correct number. Solving Sudoku is all about eliminating impossible (incorrect) numbers until some cell(s) has just one possible (correct) number.

When you have eliminated the impossible, whatever remains, however improbable, must be the truth. » Sherlock Holmes »
